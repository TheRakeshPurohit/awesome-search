# MICES

**MICES (Mix-Camp E-Commerce Search)** is a dedicated e-commerce search conference held in Berlin, timed to run alongside [[Berlin Buzzwords]] (early summer). It is community-driven and organized by [[Rene Kriegler]], with a strong practitioner focus on relevance, ranking, and query understanding for online retail. Closely overlaps with [[E-commerce Search]].

🔗 http://www.mices.co/

---

## Overview

- **Focus**: search relevance, ranking, and query understanding specifically for e-commerce.
- **Location / timing**: Berlin, adjacent to [[Berlin Buzzwords]].
- **Format**: community-driven and sponsor-supported, with pre-conference training. One edition featured a "search relevance for product managers" e-commerce training run by [[OpenSource Connections]].

## Talks in this vault

- **[[Rene Kriegler]]** — organizer; his talk [[Rene Kriegler - Query Relaxation]] reflects the MICES practitioner tradition ([[Query Relaxation]] as a query recommendation problem).
- **[[Andreas Wagner]]** — MICES talk on diverse result sets and how result positioning shapes basket composition (see [[Search Result Diversity]]).
